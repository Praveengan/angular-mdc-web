Text Field

The MDC Text Field component provides a textual input field adhering the Material Design specification.
Component
mdc-textfield
mdc-textfield-box
Properties Description
@Input() id: string Unique id of the element.
@Input() type: string = 'text' Input type of textfield (e.g. email, password, url).
@Input() name: string Name of the textfield.
@Input() value: string The input element's value.
@Input() dense: boolean Shrinks the font size and height of the input.
@Input() fullwidth: boolean Use to change element to fullwidth textfield. Not usable with mdc-textfield-box.
@Input() multiline: boolean Use to allow multiple lines inside the textfield. Not usable with mdc-textfield-box.
@Input() disabled: boolean Disables the component.
@Input() required: boolean Whether the element is required.
@Input() label: string Shown to the user when there's no focus or values.
@Input() placeholder: string Placeholder attribute of the element.
@Input() tabindex: number Tab index of the text element.
@Input() rows: number Number of rows for this textarea. Not usable with mdc-textfield-box.
@Input() cols: number Number of columns for this textarea. Not usable with mdc-textfield-box.
@Input() maxlength: number Maxlength of characters allowed to be entered.
Event Description
focus Native input element for a "focus" event.
blur Native input element for a "blur" event.
input Native input element for a "input" event.
keydown Native input element for a "keydown" event.
Directive
mdc-textfield-helptext Useful for providing supplemental information to users, as well for validation messages.
Properties Description
@Input() validation: boolean Help text can be used to provide additional validation messages.
@Input() persistent: boolean Help text will always be visible.

<mdc-textfield
  [(ngModel)]="username"
  label="Username"
  [required]="true"
  [disabled]="false"
  [dense]="false"
  (focus)="handleFocus($event);"
  (blur)="handleBlur($event);"
  (input)="handleInput($event);"
  (keydown)="handleKeyDown($event);"></mdc-textfield>

  [validation]="true"
  [persistent]="false">Username is required</p>
