Lists

MDC List provides styles which implement Material Design Lists - "A single continuous column of tessellated subdivisions of equal width." Both single-line and two-line lists are supported.
Component
mdc-list
Properties Description
@Input() dense: boolean Shrinks the font size and height of each list item.
@Input() twoLine: boolean Sets a class modifier for using two lines inside of list items.
@Input() avatar: boolean Style the start detail elements as "avatars" - large, circular details that lend themselves well to contact images, profile pictures, etc.
@Input() disableRipple: boolean Disables ink ripple. Applied at the parent mdc-list level; overrides mdc-list-item's [disableRipple].
Directive
mdc-list-item
a[mdc-list-item]
Properties Description
@Input() disableRipple: boolean
Child Directive
mdc-list-item-start Position element at the start of the row.
mdc-list-item-end Position element at the end of the row.
mdc-list-item-text
mdc-list-item-secondary
Directive
mdc-list-divider Use as full-width or inset subdivisions either within lists themselves, or event standalone.
Properties Description
@Input() inset: boolean Inset dividers are useful when working with lists which have start details.
Directive
mdc-list-group Multiple related lists can be grouped together using the mdc-list-group.
mdc-list-group-subheader Use to add a subheader to the top of your grouped list.
